Metavariable expressions used the incorrect syntax.

Metavariable expressions use curly brace and contain a single instance of a
functionlike syntax:

```rust,compile_fail,E0806
#![feature(macro_metavar_expr)]

macro_rules! foo {
    ($bar:ident) => {
        ${$bar} // invalid, no expression
        ${ignore} // invalid, no parentheses

        // invalid, cannot combine multiple expressions
        ${ignore($bar), concat($do_, $bar)}
    }
}
```

Correct usage:

```rust
#![feature(macro_metavar_expr)]

macro_rules! foo {
    ($bar:ident) => {
        ${ignore($bar)}
        ${concat(do_, $bar)}()
    }
}
```
