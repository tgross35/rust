The concat metavariable expression created an invalid expansion.

`concat` can only combine string and character literals, integer literals,
and metavariables that expand to these types.

```rust,compile_fail,E0807
#![feature(macro_metavar_expr)]
#![feature(macro_metavar_expr_concat)]

macro_rules! foo {
    ($bar:ident) => {
        ${concat(
            4,      // Identifiers cannot start with a number
            $bar,   // Ok
            4.5,    // Would not create a valid identifier
            c"foo", // C string literals are ambiguous about inclusion of `\0`
            b"bar", // Byte strings cannot be used as identifiers are UTF-8
            r#baz,  // Raw identifiers are not allowed
        )}
    }
}
```

Correct usage:

```rust
#![feature(macro_metavar_expr)]
#![feature(macro_metavar_expr_concat)]

macro_rules! foo {
    ($bar:ident) => {
        let ${concat(_, $bar, 4, "foo", baz, 'c')} = 4;
    }
}

fn main() {
    foo!(hi);
}
```
